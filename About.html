About this Website

IS 601 - Web Systems Development

New Jersey Institute of Technology NJIT

This website is the part of a Project towards credits for IS601 - Web Systems Development. This is a really nice course to take. It goes a little hectic for a few weeks of the semester, but that's when we usually learn something new. When we are pushed to cross our limits and meet the deadlines.

We really appreciate Professor Williams Keith to give us the projects that we are been given so far in this course. In the first month itself, We have learnt almost a dozen new concepts of this fast growing technology in Web development. One of our teammates, coming from a chemical engineering background, never imagined he would be able to design even a basic website without going to the core of its basics.

We really appreciate the developments made by the open-source framework bootstrap which made this possible for a non-developer background to pull this off. Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Pycharm Professional is a really nice tool to put everything in a project together. Also, linking the VCS like github to track the project for all the team members individually working on some part. It is an integrated development environment used in computer programming, specifically for the Python language. It is developed by the Czech company JetBrains.
